# Cinnagram

> **DISCLAIMER**: This project is not production ready. All versions below 1.0.0 should be considered unstable

Simple Telegram API handler

## Installation

### npm

```bash
npm install cinnagram
```

## Usage

### Send message

```javascript
sendTelegramMessage(
  "TELEGRAM_BOT_TOKEN",
  1234,
  "Hello, my *dear* friends!",
  "markdown",
);
```
